** 고객 서비스 챗봇프로젝트 과제 **

[챗봇 기본 기능]
[1] Text 질의 기능
웹  페이지  "배송문의", "환불절차", "제품 사용법","기타문의" 4개의 버튼을 클릭하여
질문 종류를 선택하고 질의 내용을 입력한 후에 입력 내용 밑에 있는 "질문하기" 버튼을 누르면
OpenAI API를 호출하여 질의를 하고 아래에 답변이 출력되도록 한다
버튼 메뉴와 질의와 답변은 모두 한글로 작성한다

[2] 이미지 업로드  질의하기
"사진 업로드" 메뉴에서 "파일 선택" 버튼을 눌러 로컬에 있는 이미지 파일을 선택해주고 아래의
"파일 업로드" 버튼을 누르면 파일이 서버로 업로드 되어 서버의 uploads 폴더에 자동으로 저장된다 
저장된 이미지를 다시 읽어와서 이미지를 base64로 인코딩하여 OpenAI API를 호출하여 질의를 하고
아래에 답변이 출력되도록 한다

[추가로 구현할 기능]
[1] 질문한 내용을 Context로 유지하면서 질문하도록 소스를 수정한다
[2] MongoDB를 사용하여 대화 내용을 저장한다. 
    (옵션)새로운 웹페이지에 서 대화 내용 조회 출력 화면 구현
[3] RAG 사용하여 "배송문의", "환불절차", "제품 사용법"을 질의하면 미리 만들어져 있는 문서를 
    참고하여 답하도록 유도한다
[4] Function Call을 사용하여 질의 시 외부 검색 (SerpAPI,Google,Bing등)함수를 호출하는 기능을 추가해본다
